# Curveapp Supplement Vendor Panel

The web dashboard for supplement store vendors on the Curve platform. Allows supplement store owners to manage their products, orders, and promotional offers.

## Tech Stack

- **Framework:** Angular 15
- **Language:** TypeScript
- **Styling:** CSS / SCSS
- **Build Tool:** Angular CLI

## Getting Started

```bash
npm install
ng serve
```

Navigate to `http://localhost:4200/`.

## Build

```bash
ng build
```

Build artifacts are stored in the `dist/` directory.
